# FloatingBall 测试用例归档

## 用例表

| 测试功能      | 预置条件                 | 输入               | 预期输出              | 是否自动 | 测试结果 |
|-----------|----------------------|------------------|-------------------|------| -------- |
| 拉起应用      | 设备正常运行               |                  | 成功拉起应用            | 是    | Pass     |
| 创建 静态布局   | 位于应用首页               | 点击【创建静态布局】       | 创建闪控球             | 是    | Pass     |
| 删除 静态布局   | 位于应用首页，已经创建闪控球         | 点击【关闭】           | 删除闪控球             | 是    | Pass     |
| 创建 普通文本布局 | 位于应用首页               | 点击【创建普通文本布局】     | 创建闪控球             | 是    | Pass     |
| 更新 普通文本布局 | 位于应用首页，已经创建闪控球         | 点击【更新】           | 更新闪控球             | 是    | Pass     |
| 删除 普通文本布局 | 位于应用首页，已经创建闪控球         | 点击【关闭】           | 删除闪控球             | 是    | Pass     |
| 创建 强调文本布局 | 位于应用首页               | 点击【创建强调文本布局】     | 创建闪控球             | 是    | Pass     |
| 更新 强调文本布局 | 位于应用首页，已经创建闪控球         | 点击【更新】           | 更新闪控球             | 是    | Pass     |
| 删除 强调文本布局 | 位于应用首页，已经创建闪控球         | 点击【关闭】           | 删除闪控球             | 是    | Pass     |
| 创建 纯文本布局  | 位于应用首页               | 点击【创建纯文本布局】      | 创建闪控球             | 是    | Pass     |
| 更新 纯文本布局  | 位于应用首页，已经创建闪控球         | 点击【更新】           | 更新闪控球             | 是    | Pass     |
| 删除 纯文本布局  | 位于应用首页，已经创建闪控球         | 点击【关闭】           | 删除闪控球             | 是    | Pass     |
| 回到应用首页    | 已经创建闪控球，当前界面在闪控球应用之外 | 点击【闪控球】          | 回到闪控球应用首页         | 否    | Pass     |
| 拖动删除      | 位于应用首页，已经创建闪控球，        | 拖动【闪控球】到回收站      | 删除闪控球             | 否    | Pass     |
| 闪控球应用分身   | 应用开启分身功能             | 创建多个【闪控球】        | 最多创建2个，最早的闪控球会被顶掉 | 否    | Pass     |
| 长按删除单个闪控球 | 应用开启分身功能，创建多个【闪控球】   | 长按【闪控球】，然后选择删除单个 | 删除单个闪控球           | 否    | Pass     |
| 长按删除所有闪控球 | 应用开启分身功能，创建多个【闪控球】   | 点击【闪控球】，然后选择删除全部 | 删除全部闪控球           | 否    | Pass     |
